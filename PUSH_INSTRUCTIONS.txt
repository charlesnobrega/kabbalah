QUICK START (3 STEPS)
================================================================================

1. Make the push script executable:
   chmod +x PUSH_TO_GITHUB.sh

2. Run the push script:
   ./PUSH_TO_GITHUB.sh

3. Done! Your project is now on GitHub.

================================================================================
MANUAL PUSH (If script doesn't work)
================================================================================

1. Initialize git:
   git init

2. Add all files:
   git add .

3. Create initial commit:
   git commit -m "Initial commit: Kabbalah specification and setup"

4. Add GitHub remote:
   git remote add origin https://github.com/charlesnobrega/kabbalah.git

5. Rename branch to main:
   git branch -M main

6. Push to GitHub:
   git push -u origin main

================================================================================
TROUBLESHOOTING
================================================================================

If you get "fatal: not a git repository":
   → Run: git init

If you get "fatal: remote origin already exists":
   → Run: git remote set-url origin https://github.com/charlesnobrega/kabbalah.git

If you get "fatal: The current branch main has no upstream branch":
   → Run: git push -u origin main

If you get "fatal: could not read Username":
   → Configure git: git config --global user.name "Your Name"
   → Configure git: git config --global user.email "your@email.com"
