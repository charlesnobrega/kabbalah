## 📌 Versionamento

Seguir [Semantic Versioning](https://semver.org/):
- **MAJOR**: Mudanças incompatíveis
- **MINOR**: Novas funcionalidades compatíveis
- **PATCH**: Correções de bugs

Formato: `MAJOR.MINOR.PATCH`

Exemplo: `1.2.3`
